# 火柴人打球小游戏
Pygame
## 项目描述
这是一个减压小游戏。玩家可以在游戏没有输赢的情况下，通过打球来释放自己的压力。

#### 玩法：
* 按”start”开始游戏
* 点人物头像选择人物并按”start”开始打球
* W键：跳； A键：左移； D键：右移； S键：发球或接球
